# Mohammad Nihal Portfolio

React portfolio for Mohammad Nihal, a Computer Science undergraduate and full-stack developer working with JavaScript, Node.js, React, AI data pipelines, and DSA.

## Tech Stack

- React
- Vite
- React Router
- Lucide React
- Custom CSS

## Structure

```text
root/
|-- index.html
|-- public/
|   |-- _redirects
|   `-- favicon.svg
|-- src/
|   |-- App.jsx
|   |-- main.jsx
|   |-- styles.css
|   |-- components/
|   |   |-- Footer.jsx
|   |   |-- Header.jsx
|   |   |-- Home.jsx
|   |   `-- ProjectDetail.jsx
|   `-- data/
|       `-- portfolio.js
`-- assets/
    `-- img/
        |-- avatar.jpg
        |-- dashboard.webp
        `-- ghosty.webp
```

## Commands

```bash
npm install
npm run dev
npm run build
```

The Netlify `_redirects` file keeps direct project routes working after deployment.
